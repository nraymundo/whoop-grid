# WHOOP Grid

A personal fitness dashboard that visualizes WHOOP recovery, sleep, and activity data
using GitHub-style calendar heatmaps.

This app helps you quickly see trends in recovery, sleep, and strain over time
in a format that is intuitive, compact, and easy to explore.

---

## Features

- Three activity grids:
  - Recovery
  - Sleep Performance
  - Strain
- Daily heatmap layout (GitHub-style)
- Hover tooltips with detailed values
- Designed for WHOOP API integration

---

## Tech Stack

- **Framework:** Next.js (App Router) + TypeScript
- **Styling:** Tailwind CSS
- **Data Source:** WHOOP API (OAuth2 authentication)
